RaBe-Wordpress-Theme
********************


0 - Information

Das RaBe-Theme ist ein Child-Theme des Wordpress-Theme-Frameworks namens Omega 
(http://wordpress.org/themes/omega) mit vielen zusätzlichen Features.


1 - Plugins

Je mehr Plugins eingesetzt werden, desto langsamer und unsicherer wird das 
Wordpress-System. Aus diesem Grund ist es ein grundsätzliches Ziel, mit dem 
RaBe-Theme ein leichtgewichtiges Theme zu realisieren, das mit möglichst 
wenigen Plugins auskommt, aber trotzdem die nötige Funktionalität bietet.

Wenn ein Plugin eingesetzt wird, so wird einerseits auf eine gute Qualität des 
Plugins und andererseits auf gute Unterstützung des Plugin-Entwicklers geschaut.


2 - Benutzer

Grundsätzlich hat jeder Benutzer eine klare Rolle. Der Benutzer "superadmin" 
sollte nur in seltenen Fällen benutzt werden.


3 - Interface

Das Wordpress-Interface (im Hintergrund) wurde für die Gruppe der 
Sendungsmachenden sowie weitere Rollen mit weniger Rechten stark vereinfacht, 
so dass sich auch weniger Computer affine Menschen zurecht finden.
